## ⚙️ Instalação

```
# Abra um terminal e copie este repositório com o comando
$ git clone https://github.com/Hugovarellaa/ignite-feed
```

```
# Acesse a pasta da aplicação
$ cd ignite-feed
$ cd ignite-feed-typescript 

ou

$ cd ignite-feed 

# Instale as dependências
$ yarn

# Inicie a aplicação
$ yarn dev

```
